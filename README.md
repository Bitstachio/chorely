# Chorely

Chorely is a simple to-do list application built with React, allowing users to add, edit, and delete tasks.

## Prerequisites

- [Node.js](https://nodejs.org/) (Recommended: Latest LTS version)
- [npm](https://www.npmjs.com/) (Comes with Node.js)

## Installation

### Clone the Repository

```sh
git clone https://github.com/Bitstachio/chorely.git
```

### Install Dependencies

```sh
npm install
```

### Run the Project (Dev)

```sh
npm run dev
```

### Run the Tests

This project uses `Vitest` as the testing framework along with `React Testing Library` for component testing and
`jest-dom` for additional matchers.

```sh
npm run test
```
